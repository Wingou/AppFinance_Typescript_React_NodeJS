# financereact
App Finance in React
15/02/2025

========================
EXECUTION SERVER + APPLI
========================
$> node ./api/server.js
$> npm start

========================
Create REACT environment
========================
Create REACT environment in ROOT :
$> npx create-react-app financereact

========================
SERVER
========================
Create folder .\api
Copy these files :
.\.env
.\api\server.js
.\api\utils.js
.\api\queries.js

Installing modules :
$> npm install odbc
### $> npm install dotenv


========================
 APPLI
========================

EXECUTION :
$> npm start
$> npm test
$> npm run build
